bookstore 
======
One-page application 
 
### Used technologies:

- JavaScript - jQuery(Ajax)
- PHP
- Bootstrap

### Possible Operation:  
- add, delete, edit, and display books from the database
- use methods: PUT, DELETE, GET, POST 
- use JSON format to send and receive data
- use jQuery to add and remove elements of DOM structure
